[TicTacToe.cz](https://www.tictactoe.cz) je open-source webová aplikace s 3 hrami (v tento moment) - Tic Tac Toe, Snake a Tetris. Tetris si můžete zahrát v singleplayer i multiplayer módu.

Aplikace je postavená na Next.js a Firebase. Můžete si projít její [otevřený GitHub repozitář](https://github.com/cibulka/tictactoe.cz).

## Instalace

Naklonujte si repo, nainstalujte knihovny a spusťte to.

```bash
git clone https://github.com/cibulka/tictactoe
cd tictactoe
npm install
npm run dev
```

V základním nastavení aplikace běží na [http://localhost:3000](http://localhost:3000).
