[TicTacToe.cz](https://www.tictactoe.cz) is an open-source web app with 3 games (at the moment) - Tic Tac Toe, Snake and Tetris. You can play Tic Tac Toe in singleplayer or multiplayer mode.

It is built with Next.js and Firebase. You can check its [open GitHub repository](https://github.com/cibulka/tictactoe);

## Installation

Clone the repo, install the dependencies and run it.

```bash
git clone https://github.com/cibulka/tictactoe
cd tictactoe
npm install
npm run dev
```

By default, the app will run at [http://localhost:3000](http://localhost:3000).
